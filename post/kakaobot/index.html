Golang kakaotalk chat bot making

Posted on March 15, 2017

Go 언어로 간단한 Kakaotalk chat bot 을 만들어 봤습니다. Go 언어의 강력함과 Kakaotalk auto-reply API 의 단순성 덕에 두시간 만에 만들어지더군요. 일단은 단순한 echo 서버 정도의 역할만 하고 있습니다만, 매우 단순하게 짜여져 있기 때문에 참고하면 Kakaotalk chat bot 을 만들고자 할 때, 또는 Restful API 를 사용하는 서버를 Go 언어로 짜고자 할 때 참고할 만 하지 않을까 싶어 소개합니다.
